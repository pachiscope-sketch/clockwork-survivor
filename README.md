# Clockwork Survivor

Unity実績づくり向けの小型プロジェクトです。歯車仕掛けの闘技場で3分間生き残る、見下ろし型ローグライト風サバイバルゲームとして作っています。

## 遊び方

- `WASD` / 矢印キー: 移動
- マウス位置: プレイヤーの向き
- 攻撃: 最寄りの敵へ自動射撃
- 経験値を集めてレベルアップすると、3つの強化から1つを選択

## 実装済み

- タイトル、ゲーム中HUD、強化選択、リザルト画面
- プレイヤー移動と自動攻撃
- Walker / Dasher / Shooter の3種類の敵
- 敵スポーンと時間経過による難度上昇
- HP、経験値、レベルアップ、強化選択
- ScriptableObjectによる敵、武器、強化データ
- WebGLを意識した軽量なBuilt-in構成

## 初回セットアップ

このプロジェクトをUnity 6000.3.2f1以降で開くと、`Assets/Scenes/ClockworkSurvivor.unity` が自動生成されます。

自動生成されない場合は、Unityメニューから次を実行してください。

`Clockwork Survivor > Setup Project`
